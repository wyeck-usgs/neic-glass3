Detailed Description

traveltime branch class

The traveltime CTrav class is a class that encapsulates one or more 1D interpolated distance/time spline functions representing seismic phase branches at a specific depth.

The CTrav class uses the distance/time spline functions to calculate the expected travel times and distances. The spline functions are generated from the provided CRay parameters.
