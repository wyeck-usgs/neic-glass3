Detailed Description

namespace containing the glass core library

The glasscore namespace contains the primary glass core library, including all the classes and components necessary for association and nuclation of new events.

Performs a Brent Minimization of the function

Given a function f, bracketing intervals, and a stopping tolerance compute the minimum of the function, and the location of that minimum. The Notation and implementation based on Chapter 5 of Richard Brent's book "Algorithms for Minimization Without Derivatives".

Parameters
f- An objective function of type TFunction to minimize
leftEnd- A Double value containing the smaller bracketing interval
rightEnd= A Double value containing the larger bracketing interval
epsilon- A Double value containing the stopping tolerance
minLoc- A Double reference to return the location where f takes its minimum
Returns
Returns a double value containing the minimum of the function
