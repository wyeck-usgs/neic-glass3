Detailed Description

glassutil geographic coordinate conversion class

The glassutil CGeo class is a class that encapsulates the logic necessary to convert geographic, geocentric, and Cartesian coordinates to geographic and geocentric coordinates. The class also computes the distance and azimuth between coordinates, and intersections between geographic (or geocentric) spheres.
