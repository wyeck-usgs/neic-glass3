Detailed Description

traveltime ray path class

The traveltime CRay class is a class that encapsulates calculating travel time ray parameters, and using the ray parameters to calculate expected travel times and distances. The Ray parameters are calculated via inversions over a provided CTerra earth structure model.

Note that as written, CRay is NOT thread safe.
